# Introdução sobre programação

## Conceitos aprendidos:

* Algoritmos
    É uma sequência finitas de instruções para resolver um Problema.

* Automoção
    É quando uma máquina realiza o algoritmo.

* Programa de Computador
    È um algoritmo executado pelo computador.

* O que eu preciso para fazer um Programa?
    1ª Escolher uma Linguagem de programação; 
    2ª Escolher uma IDE especificar dessa linguagem;
    3ª Um Compilador;
    4ª Um gerador de código ou máquina virtual.
